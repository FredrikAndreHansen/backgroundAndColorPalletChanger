A PowerShell script that randomly changes the background image and selects a color accent.

The background image is immediately changed.
The color accent is manually changed by the script, meaning that it opens settings and navigate to the desired color, selects the color and then closes settings.

#######################################

Configuration

In program.ps1:
The "$FullPath" variable is the path where the script is located
"$allImages" is an array that contains the image names, it is configured to be located in the same folder as the script
